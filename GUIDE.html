Overview

Excel Workbook Manager is a standalone Windows desktop tool that sits alongside Microsoft Excel and automates the repetitive, error-prone workbook management tasks that arise during actuarial period-end processes — IFRS 17 reporting, Solvency II QRTs, board packs, and similar workflows.

The tool connects to Excel via COM automation (the same mechanism that powers VBA macros), so every action it takes is equivalent to what you could do yourself inside Excel — just much faster and with a permanent audit trail.

No installation needed. The tool ships as a self-contained ExcelWorkbookManager.exe. Place it anywhere on your PC and double-click to run. No Python, no admin rights, no Office add-in registration.

Quick Start

  1. Open Excel and load your workbooks The tool only sees workbooks that are currently open in Excel. Open everything you plan to work with before launching.
  2. Double-click ExcelWorkbookManager.exe A brief splash screen appears while the tool connects to Excel. The main window opens automatically with the last active tab restored.
  3. Check the Excel Instance picker (bottom bar) If you have multiple Excel windows open, select the right instance from the Excel: dropdown, or leave it on [ALL INSTANCES] to see everything.
  4. Click Refresh All Lists This re-scans all open workbooks. Use it any time you open or close a file in Excel. The tool also auto-refreshes every 20 seconds in the background.
  5. Select a tab and work Each tab is self-contained. Start with your most common task — for period-end work this is usually Smart Relinker → Tab Refresh → Smart Refresh.

1 Close Workbooks

Lists every open Excel workbook and lets you close any number of them in bulk. Useful at the end of a period-end run when you need to tidy up a large set of open files without closing Excel itself.

How to use it

  1. Click Refresh Lists (or wait for the auto-refresh) The list populates with every open workbook, its sheet count, and whether it has unsaved changes.
  2. Use the Search box to filter Type any part of the workbook name. The list updates in real time (200 ms debounce). Clear the box to show all.
  3. Select workbooks to close Click individual rows, or use Select All / Deselect All. You can also Shift+Click or Ctrl+Click for multi-select.
  4. Click Close Selected If any selected workbook has unsaved changes, a 3-button dialog appears: Save & Close, Close Without Saving, or Cancel. Your choice applies to all selected workbooks in that batch.

Columns explained

ColumnMeaning
Workbook NameThe filename as Excel knows it. Shown in orange if the workbook has unsaved changes.
SheetsTotal number of sheets (visible + hidden).
Unsaved?"Yes" if the workbook's Saved property is False. Rows marked "Yes" are highlighted in orange as a visual warning.

Right-click menu

Right-click any row to access: Close Selected, Copy to Clipboard, Export to CSV.

Tip: The list does not include PERSONAL.XLSB — the personal macro workbook is always excluded to prevent accidental closing.

2 Open Links (Link Explorer)

Displays all external workbook links stored in a selected workbook and lets you open, verify, or permanently break them.

Listing links

  1. Select a workbook from the Source Workbook dropdown
  2. Click List Links All external .xls* paths registered in the workbook's link table are shown.
  3. Optionally tick "Check file existence (may be slow on network drives)" When ticked each link's status is resolved: OK (file found on disk), BROKEN (file not found), or Unknown (UNC/network path could not be reached). Leave unticked on network drives to avoid slowdowns.

Opening links

Select one or more links in the list, then choose one of the two open buttons:

ButtonWhat opens
Open Selected Opens every highlighted link. After opening, the tool calls UpdateLink + CalculateFullRebuild on the source workbook so formulas immediately pick up live values from the newly opened files.
Open Current Tab's Links (Ctrl+Shift+O) Scans the active sheet in the selected workbook for formula references of the form 'path\[Book.xlsx]Sheet'!cell and opens only those. Does not open links used only on other sheets.

The Open as Read-Only checkbox (on by default) controls whether linked workbooks open read-only or read-write.

Breaking links

Select links, click Break Selected Links, and confirm. Breaking a link calls Excel's Workbook.BreakLink which replaces every formula referencing that file with its last-calculated static value. This cannot be undone with Ctrl+Z — save the workbook first if you are unsure.

Status colours

ColourMeaning
RedBROKEN — file not found on disk.
GreenOK — file exists.
GreyStatus not checked (default when existence-check is off).
Ctrl+Shift+O scope: This shortcut only detects links written as direct formula references. Links from Power Query, defined names, or DDE connections are not detected. Use List Links + Open Selected for those.

3 Tab Refresh

Forces Excel to recalculate specific sheets — bypassing the global calculation mode. Useful when you want to push a handful of output sheets without recalculating the entire workbook.

Options panel

OptionDescription
Workbook The workbook whose sheets you want to refresh.
Refreshes Number of calculation passes per sheet. Use 2 or more when iterative calculation is needed (e.g. circular references or sequential dependency chains within the sheet).
Order Normal = top-to-bottom as listed; Reverse = bottom-to-top. Useful when later sheets feed back into earlier ones.
Change log See table below.

Change log options

SettingWhat happens
No Refresh runs silently — no log produced.
New File After the refresh a Save dialog appears. Every cell whose value changed is written to a colour-coded .xlsx file with columns: Tab Name, Cell Address, Original Value, Changed Value.
Add Tab to Workbook A new sheet named ChangeLog_MMDD_HHMMSS is appended directly to the source workbook. The cell addresses in the log correctly reflect the UsedRange origin — a cell at row 3 col 2 is reported as $B$3, not $A$1.

Refresh All Connections button

This separate button calls Workbook.RefreshAll() on the selected workbook. This refreshes all Power Query queries, pivot caches, and ODBC/database connections in one go — equivalent to clicking Data → Refresh All in Excel.

Workflow tips

  • Click List Sheets first to populate the sheet list, then select only the sheets you need before clicking Refresh Selected Tabs.
  • Use Cancel (appears while running) to stop after the current sheet completes.
  • For a full model refresh, use the Smart Refresh tab instead — it handles link-opening and audit-cell comparison as well.

4 Smart Relinker

Redirects the external workbook links stored inside one or more open workbooks. This is the primary tool for the quarterly period roll — changing \2025\202512\ links to \2026\202603\ across dozens of files.

Four modes are available via radio buttons at the top. Each mode fills the same Preview grid at the bottom — all modes are non-destructive until you click Apply Now.

① Period Shift mode

Enter an Old Period and a New Period (6-digit YYYYMM). The tool replaces:

  • The year subfolder — e.g. \2025\ → \2026\
  • Every occurrence of the period token in the path — e.g. 202512 → 202603
Before: I:\IFRS 17\2025\202512\Working\RI Hub\RI Hub AoM 25FY.xlsx After: I:\IFRS 17\2026\202603\Working\RI Hub\RI Hub AoM 25FY.xlsx

File-name suffixes like 25FY are not changed by Period Shift alone. Add suffix rename rules in the Extra Rename Rules list, or load a quarterly profile (see Config Profiles).

② Custom Find & Replace mode

Define a list of find/replace text pairs applied in sequence to every link path. Good for ad-hoc corrections, non-standard folder restructuring, or standalone suffix changes without a period shift.

  • Rules are applied top-to-bottom — order matters if one rule's output is the input for the next.
  • Rules can be individually enabled or disabled with the checkbox on each row.
  • The rule list can be saved as a profile for re-use.

③ Folder Match mode (fuzzy scan)

Use this when the folder layout or file names have changed in ways you cannot describe with a simple find/replace — e.g. the vendor renamed files, the subfolder tree was restructured, or the period token moved position.
  1. Select the new root folder Click the folder browser and choose the top-level directory under which the new files live.
  2. Set the minimum score threshold Default is 70%. Lower it (e.g. 50%) if you get too many "No Match" rows; raise it (e.g. 85%) if wrong files are auto-matched.
  3. Click Scan & Match The tool walks the folder recursively, indexes every .xls* file, and computes a fuzzy similarity score for each old link vs every candidate.

How the fuzzy score is calculated

Each filename is reduced to a skeleton before scoring: 8-digit dates, 6-digit period tokens, and version strings (v1.2, _v3) are stripped. The skeleton is lower-cased and spaces/underscores are normalised. Python's difflib.SequenceMatcher then compares skeletons, producing a 0–1 score.

RI Hub AoM 25FY 20251231 v1.2.xlsx → strip date, version → "RI Hub AoM 25FY" → lower + normalise → "ri hub aom 25fy" ← skeleton for scoring

Match outcomes

OutcomeWhat you see
One strong match Auto-filled in preview with tag Folder Match (92%).
Multiple close matches A picker dialog shows all candidates with scores, same-folder bonus indicator, and full paths. Double-click or click Use Selected to choose; Skip leaves the row unchanged.
No match ≥ threshold Row stays unchanged and is tagged No Match for manual correction.
  • Files in the same relative subfolder as the original link are boosted above equal-scoring files in different folders.
  • Cross-extension matches (.xlsm → .xlsx) are penalised 5%.
  • The scan is purely file-system based — no COM calls, so it's very fast (10,000-file trees typically scan in a few seconds).

④ Manual Edit mode

Loads all links into the preview exactly as they are — no rules applied. Select any row, type a corrected path in the Manual Override box, click Update Row, then Apply Now. Use this for one-off corrections that are too varied for any automated rule.

The Queue System

The queue lets you prepare relink changes across multiple workbooks and apply them all in a single final step:

  1. Set up rules and generate a preview for workbook A
  2. Click Queue All Modified Every row in the preview where the New Path differs from the Current Path is captured as a named queue entry (e.g. "Book A — 12 pairs").
  3. Switch to workbook B, generate preview, Queue All Modified again Repeat for as many workbooks as needed.
  4. Click Apply Queue All entries are processed in order. A summary log appears on completion.

Queue entries can be removed individually before applying.

Preview Grid — shared by all modes

The preview is the "commit surface" — it shows exactly what will change before you commit. It is fully editable.

ColumnMeaning
WorkbookSource workbook that owns this link.
Current PathThe link as stored in Excel right now.
New PathThe computed target path. Edit this cell directly for a manual override.
MethodHow the new path was generated: Period Shift, Find/Replace, Folder Match, Manual.
✔/✘File-exists check result (shown when "Verify new paths exist" is ticked).
  • Open in Excel — send the full Before/After table into a live spreadsheet. Edit the yellow New Path column in Excel cells, then click Import from Excel to pull the edits back.
  • Keep Verify new paths exist ticked. If a target file is missing, the tool warns you before writing any changes.

5 Period Rollover Wizard

Rolls all selected open workbooks from one reporting period to the next in a single guided step — applies Period Shift rules, rename rules, and optionally Tab Refresh, without having to visit the Smart Relinker for each file individually.

  1. Select workbooks All open workbooks are listed. Deselect any you want to skip (e.g. data extract files that do not carry period-specific links).
  2. Set Old Period and New Period, or load a profile Enter 6-digit YYYYMM codes, or select a built-in quarterly profile from the dropdown (which pre-fills both periods and all rename rules).
  3. Review / add Extra Rename Rules e.g. 25FY → 26Q1. These are already included if you loaded a standard quarterly profile.
  4. Click Preview All Changes Review the full Before/After list across every selected workbook before committing.
  5. Click Apply Rollover All relinks are applied. If Run Tab Refresh after relink is checked, every sheet in every affected workbook is recalculated automatically.
When to use Rollover vs Smart Relinker: Use the Rollover wizard when all workbooks follow the same period naming convention and a single set of rules applies to all. Use Smart Relinker when workbooks have different conventions and need per-file customisation.

6 Dependency Map

Builds an expandable tree showing which workbook links to which other workbook or external file. Useful for understanding refresh order and for documentation.

Building the map

  1. Set Max Depth (1–6, default 2) Depth 1 shows only direct links from each open workbook. Depth 2 also shows what each of those targets links to (if open), and so on.
  2. Optionally tick "Recurse into open workbooks" When ticked, the tree follows links through any target that is currently open in Excel, up to the Max Depth limit.
  3. Click Build Dependency Map

Reading the tree

ColourMeaning
GreenThe linked file is currently open in Excel.
GreyThe linked file is external (not open).
Light greyNo outgoing links from this workbook.

The # Links column shows how many cells in the source workbook reference each target file. Use Expand All / Collapse All for quick navigation.

Exporting

Export to Excel saves the full map as a formatted .xlsx with columns: Source Workbook, Links To, Full Path, # Links, Status. Useful for documentation, sign-off, or planning the recalculation order before a period-end run.

Staleness notice: If you open or close workbooks after building the map, a warning banner appears. Click Build Dependency Map again to refresh.

7 Error Scanner

Scans all sheets in a workbook for formula error values and presents the full list with sheet, address, error type, and formula text.

  1. Select a workbook and click Scan for Errors A progress bar shows which sheet is being scanned and how many are remaining. Click Cancel to stop after the current sheet.
  2. Use the Filter box to narrow results Type any text — the list is filtered instantly across Sheet, Cell, Error, and Formula columns simultaneously.
  3. Select a row and click Go To Cell Excel activates the sheet and selects the cell. You can also double-click a row (right-click menu).

Use Export to CSV or Copy to Clipboard to share the error list with reviewers. Right-click any row for the same options.

8 Sheet Navigator

A searchable master list of every sheet across all open workbooks. Navigate to any sheet instantly without scrolling through Excel's tab bar.

Searching

Two independent filter boxes:

  • Workbook: narrows by workbook name (case-insensitive substring).
  • Sheet: narrows by sheet name.

Both filters are applied as AND conditions — e.g. Workbook "RI Hub" AND Sheet "AoM" shows only sheets named "AoM" that belong to workbooks with "RI Hub" in their name.

Hidden sheets

By default the list includes hidden and very-hidden sheets, shown in grey and red respectively. Untick Show hidden sheets to hide them.

ColourVisibility
BlackVisible — normal sheet.
GreyHidden — xlSheetHidden. Can be unhidden from the Excel tab bar context menu.
RedVery Hidden — xlSheetVeryHidden. Cannot be unhidden from the Excel UI — VBA or this tool required.

Visibility actions

Select a sheet and use the action buttons:

ButtonAction
Make VisibleSets xlSheetVisible. Works on Hidden and Very Hidden sheets.
HideSets xlSheetHidden. Reversible from Excel's UI.
Very HideSets xlSheetVeryHidden. Only reversible via VBA, the tool, or the Name Manager trick.
Go To SheetActivates the sheet in Excel. Also triggered by double-clicking a row.

9 Volatile Scanner

Finds cells containing volatile functions — functions that force Excel to recalculate the entire workbook every time any cell changes, regardless of whether they have any dependency on what changed.

Volatile functions detected

INDIRECT OFFSET NOW TODAY RAND RANDBETWEEN INFO CELL

The scanner uses word-boundary matching: NOW() is flagged, but NOWHERE or a quoted string containing "NOW" is not.

Impact rating

CountRatingRecommendation
0None — Clean!No action needed.
1–20LowAcceptable in most models.
21–100MediumConsider replacing INDIRECT/OFFSET with direct references where possible.
100+HighLikely causing significant slowness. Prioritise elimination of INDIRECT/OFFSET in inner-loop formulas.

Use the live Filter box to narrow to specific functions or sheets. Click Go To Cell (or double-click) to jump to any result.

Listing

11 Find & Replace

Searches for text across a whole workbook or a single sheet, and optionally replaces all matches at once. Results are navigable — double-click any row to jump directly to that cell in Excel.

Search options

OptionDescription
Workbook The workbook to search.
Sheets All searches every sheet. Select a specific sheet name to limit the scope.
Find The text to search for.
Replace The replacement text (used only by Replace All).
Match case Case-sensitive matching.
Match whole cell Only match if the entire cell content equals the search term.
Search in formulas Search within formula text rather than displayed values. Useful for finding cell references, function names, or link paths embedded in formulas.

Finding

Click Find All. Results appear in the list with columns: Sheet, Address, Value, Formula. The search runs in a background thread — a progress bar is shown while it runs.

12 Smart Refresh

The actuarial-specific automation tab. It combines four steps that are normally done manually at every period-end refresh: open linked documents, recalculate sheets, compare key figures, and log the changes for audit. All of this can be scheduled to run automatically every N minutes.

Four-panel layout

PanelPurpose
Workbook & Profile (top)Choose the source workbook; save/load named run profiles.
Linked Docs to Pre-Open (left)List of all external links discovered in the workbook. Tick which ones to open before each refresh.
Sheets to Refresh (right)Select which sheets to recalculate on each run.
Audit Cells (middle)Define the key figures to compare before and after each run.

Step-by-step setup

  1. Select the source workbook and click Discover Links All external .xls* paths are listed in the left panel. Tick every linked document that must be open for the refresh formulas to update correctly. Untick files that are always open or that you manage separately.
  2. Click Load Sheets in the right panel and select the sheets to recalculate e.g. your output summary sheet and reconciliation tabs.
  3. Define Audit Cells For each key figure you want tracked:
    • Label — a descriptive name (e.g. "BEL Reserve Total")
    • Sheet — choose from the dropdown (populated after Load Sheets)
    • Address — cell address like C5 or a range like B3:D10
    Click Add Cell. Add as many as needed.
  4. Click Run Now or set up a schedule See below.

Run Now — what happens in order

  1. Read the current value of every audit cell → BEFORE snapshot
  2. Open any checked linked documents that are not already open (read-only)
  3. Call Application.Calculate on each selected sheet
  4. Call UpdateLink + CalculateFullRebuild on the source workbook
  5. Read the current value of every audit cell → AFTER snapshot
  6. Append one row per audit cell to the Audit Log

Scheduled auto-run

Set the Auto-run every N minutes spinbox, then click Start Schedule. A countdown label shows "Next run in 4:32" and counts down second by second. Click Stop Schedule to cancel.

Important: The schedule is purely in-app — it only runs while the tool window is open. It is not a Windows Task Scheduler job. Minimise the window to let it run in the background while you work elsewhere.

Audit Log

The log at the bottom shows every audit-cell comparison from every run in the session.

ColumnMeaning
TimestampDate and time the run completed.
WorkbookSource workbook name.
LabelThe label you gave this audit cell.
Sheet / AddressWhere the value was read from.
Before / AfterValue before and after the refresh. For multi-cell ranges, the first 8 values are shown comma-separated.

Export the log with Export to Excel (colour-coded .xlsx), Export to CSV, or Copy to Clipboard. Use Clear Log to start fresh.

Profiles — saving and reusing configurations

Save the entire setup (workbook, checked links, selected sheets, audit cells, interval) as a named profile. Profiles are stored in settings.json and persist across sessions.

  • Click Save Profile — enter a name (e.g. "IFRS17 Month-End BEL", "Solvency QRT 202603").
  • To reload a profile: select its name from the Profile dropdown and select it. All panels restore automatically.
  • Click Delete Profile to remove a profile permanently.
Tip: Create one profile per report type. Before each period-end run, load the profile for that report, click Run Now, and your audit log is automatically populated.

13 Snapshot Export

Creates a formula-free copy of selected sheets. Every formula cell is replaced by its current displayed value (equivalent to Copy → Paste Special → Values in Excel). The output file is named automatically with the date suffix.

When to use it

  • Send a "locked" version of model outputs to reviewers who should not see formulas.
  • Create a dated period-end archive: BEL_Model_20260331.xlsb.
  • Reduce file size — binary .xlsb with values only can be 50–70% smaller than the formula-bearing original.
  • Produce distribution workbooks that open instantly because there is nothing to recalculate.

Step-by-step

  1. Select the source workbook and click Load Sheets All sheets are listed and all are selected by default.
  2. Deselect sheets you do not want in the snapshot e.g. internal helper tabs, parameter sheets, or VBA-only sheets.
  3. Choose or confirm the Save Folder Defaults to the same directory as the source workbook. Click Browse… to change it.
  4. Set the Suffix type
    • Date (YYYYMMDD) — e.g. Budget_Model_20260430.xlsb
    • Custom — type any suffix: _MonthEnd, _v2_final, _ForReview
  5. Choose the output Format .xlsb (smallest, binary), .xlsx (most compatible), .xlsm (if you need macros), or .xls (legacy).
  6. Set the optional checkboxes
    • Save source workbook first — saves the original before exporting, so the snapshot reflects the latest saved state.
    • Open in Excel after saving — opens the new file immediately for review.
  7. Check the filename preview and click Export Snapshot The preview label above the button shows the exact full path before you commit. If the file already exists, a confirmation dialog appears before overwriting.

What is and is not affected

PreservedConverted to values
Cell formatting, fonts, colours, borders All formula cells → static values
Charts and chart data (Charts still display correctly; the underlying data cells become values)
Pivot tables (structure) Pivot table source formulas if in the UsedRange
Sheet names, tab colours, print areas
Named ranges scoped to the sheet
File size tip: For the smallest output, choose .xlsb and deselect any sheets that are purely data-input tabs (no visible outputs). A 500 MB formulaic model often reduces to under 50 MB as a values-only .xlsb.

Workbook Health

Runs a background diagnostic scan across all open workbooks and produces a summary table with health indicators.

How to run

Click Refresh Dashboard. The scan runs in a background thread so the tool remains responsive. A progress bar is shown while scanning.

The dashboard does not auto-run on every refresh — it only updates when you explicitly click the button. If you open or close workbooks after scanning, a staleness warning appears at the top.

Adding custom profiles

Open profiles.json in any text editor and add an entry:

{ "name": "My Custom Roll", "description": "Optional note", "period_shift": { "old": "202512", "new": "202603" }, "rules": [ { "find": "25FY", "replace": "26Q1" }, { "find": " FY", "replace": " Q1" } ] }

A profile with only rules and no period_shift is valid — useful for pure find-and-replace scenarios.

Tips & Notes

  • Always Refresh All Lists after opening or closing workbooks in Excel. The tool auto-refreshes every 20 seconds but this may lag by up to 20 seconds.
  • Preview before applying in the Smart Relinker. The preview is fully editable and shows exactly what will change. Never click Apply Now without reviewing the preview first.
  • Verify new paths exist — keep this checkbox on in the Relinker. If a target file is not found on disk you are warned before any link is rewritten.
  • Folder Match threshold — default 70%. Lower to 50–60% if you get too many "No Match" rows; raise to 80–90% if wrong files are auto-matched.
  • Queue All Modified in the Relinker lets you prepare changes for multiple workbooks before applying them all in one go — useful for batch period rolls.
  • Smart Refresh profiles — create one profile per report type (BEL, RA, QRT, etc.). Each profile stores the workbook, linked docs, sheets, audit cells, and interval. Loading a profile takes two seconds; running the full refresh takes as long as Excel needs to recalculate.
  • Snapshot Export + date suffix — run Snapshot Export at the end of each period and you automatically build a dated archive trail: Model_20260331.xlsb, Model_20260630.xlsb, etc.
  • .xlsb format is recommended for snapshots and archives. It is the most compact Excel format and opens significantly faster than .xlsx for large files.
  • Dark Mode — click the moon icon in the bottom bar. Your preference is saved and restored on next launch.
  • Hide tabs you don't use via the ⚙ Settings button. Hidden tabs remain fully functional — they just don't appear in the tab bar. Restore them at any time.
  • profiles.json — edit this file in Notepad to add custom quarterly rollover rules. Changes take effect next time you open the Relinker or Rollover tab.
  • Break Links is irreversible within the same session — save the workbook first before breaking any links.
  • Replace All in Find & Replace is not Ctrl+Z-able across all sheets — save the workbook first.
