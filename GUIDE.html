Overview

Excel Workbook Manager is a standalone Windows desktop application that sits alongside Excel and automates repetitive workbook management tasks — particularly those that arise during period-end actuarial processes such as IFRS 17 reporting.

Rather than manually hunting through Excel menus, it provides a unified control panel to close workbooks, relink external references, refresh calculation tabs, scan for errors, and much more — all without touching a macro or VBA editor.

No installation required. The tool ships as a single .exe file. Copy it to any Windows machine that has Excel installed and double-click to run.

Requirements

  • Windows 10 or 11
  • Microsoft Excel (desktop, any recent version)
  • Excel must be open and running before launching the tool
  • No Python installation needed on the target machine
Important: Always open Excel first, then launch the tool. If Excel is not running you will see a startup error.

Quick Start

  1. Open Excel Load the workbooks you want to work with — the tool reads whatever is currently open.
  2. Launch ExcelWorkbookManager.exe A splash screen appears briefly; the main window opens automatically.
  3. Click "Refresh All Lists" The bottom-left button re-scans all open workbooks. Use it any time you open or close a file in Excel.
  4. Select a tab and work Each tab is self-contained. Hover over ℹ buttons for inline explanations.

Close Workbooks

Lists all open workbooks with size and path. Select one or many and close them in bulk. Prevents the "close one-by-one" routine when finishing a period.

Link Explorer

Scans all open workbooks and displays every external link (source path, type, status). Quickly identify broken, missing or unexpected references.

Select individual sheets and run multiple recalculation passes in any order. Optionally tracks cell-level changes and saves a change log — either to a new Excel file or directly as a new tab in the source workbook.

Smart Relinker

Redirect external links using Period Shift (automatic year + period token replacement), multi-rule Find & Replace, or a saved profile. Includes a live Excel-grid preview for manual path editing before applying.

⚡
Period Rollover

One-click wizard to roll all selected workbooks from one period to the next. Applies period shift + rename rules + optional Tab Refresh in a single operation.

🗺
Dependency Map

Builds an expandable tree showing which workbook links to which. Identifies open vs external files. Exportable to Excel for documentation.

⚠
Error Scanner

Scans selected sheets for formula errors (#REF!, #N/A, #VALUE! etc.) and lists every affected cell. Jump directly to any flagged cell from the results list.

📋
Sheet Navigator

Searchable list of every sheet across all open workbooks. Click any row to activate that sheet instantly — no more scrolling through tab bars.

🔍
Volatile Scanner

Locates cells using volatile functions (NOW, TODAY, RAND, INDIRECT, OFFSET etc.) that force full recalculation on every change — a common source of slowness.

📌
Named Ranges

Lists all named ranges and their references across open workbooks. Filter, inspect and jump to any named range in one click.

❤
Workbook Health

Runs a multi-check diagnostic (broken links, error cells, volatile functions, external references, hidden sheets) and produces a traffic-light summary report for each workbook.

Smart Relinker — How It Works

The Smart Relinker is designed around the IFRS 17 folder convention where both the year folder and the period token in file names change every quarter:

I:\IFRS 17\2025\202512\Working\Non-model\RI Hub\RI Hub AoM 25FY.xlsx ↓ Period Shift: 202512 → 202603 + Rename rule: 25FY → 26Q1 I:\IFRS 17\2026\202603\Working\Non-model\RI Hub\RI Hub AoM 26Q1.xlsx

Period Shift mode

Enter an Old Period and a New Period (6-digit YYYYMM). The tool automatically replaces:

  • The year folder — e.g. \2025\ → \2026\
  • Every occurrence of the period token — e.g. 202512 → 202603

File name suffixes like 25FY or FY are not changed by the period shift alone — add them as Extra Rename Rules in the rules list below the mode selector.

Custom Find & Replace mode

Define a list of Find / Replace pairs applied in sequence to every selected link path. Useful for one-off corrections or non-standard folder structures.

Open in Excel preview

Before applying, click Open in Excel to see the full Before / After table in a live spreadsheet. Edit the yellow New Path column directly in Excel cells, then click Import from Excel to bring the edits back.

Period Rollover Wizard

Use this tab when you need to roll multiple workbooks at once from one reporting period to the next. It combines relinking, rename rules, and tab refresh in a single guided flow.

  1. Select workbooks All open workbooks are listed and pre-selected. Deselect any you want to skip.
  2. Set periods or load a profile Enter Old Period and New Period, or pick a saved profile from the dropdown (which pre-fills both periods and all rename rules automatically).
  3. Add extra rename rules if needed e.g. 25FY → 26Q1. These are already included if you loaded a standard quarterly profile.
  4. Preview All Changes Review the full list of Before / After paths across every selected workbook.
  5. Apply Rollover Applies all relinks. If Run Tab Refresh after relink is checked, every sheet in every affected workbook is recalculated automatically.

Config Profiles

Profiles save a complete set of relinking rules (period shift + rename rules) so you never need to re-enter them each quarter. They are stored in profiles.json alongside the .exe file.

Q1 2026 → HY 2026 202603 → 202606 26Q1→26HY  ·   Q1 → HY

Adding your own profiles

Open profiles.json in Notepad or any text editor. Each profile follows this structure:

{ "name": "My Custom Roll", "description": "Optional note about this profile", "period_shift": { "old": "202512", "new": "202603" }, "rules": [ { "find": "25FY", "replace": "26Q1" }, { "find": " FY", "replace": " Q1" } ] }
Tip: You can have a profile with only rules and no period_shift block — useful for pure find-and-replace scenarios.

Workbook Dependency Map

Click Build Dependency Map to scan all open workbooks and produce an expandable tree:

  • Green entries — the linked file is also currently open in Excel
  • Grey entries — the linked file is external (not open)
  • The # Links column shows how many cells in the source workbook point to that target

Use Export to Excel to save the full map as a formatted spreadsheet — handy for documentation or reviewing the refresh order before a period-end run.

Tab Refresh — Change Log Options

When Change log is set to anything other than No, the tool records every cell whose value changed during the refresh.

SettingWhat happens
No Refresh runs silently with no log.
New File After refresh, a Save dialog appears and the log is saved as a new colour-coded .xlsx file.
Add Tab to Workbook A new sheet named ChangeLog_MMDD_HHMMSS is added directly to the source workbook and activated automatically — no extra file created.

Tips & Notes

  • Dark / Light mode — click the 🌙 Dark Mode button in the bottom bar to toggle between themes instantly.
  • Column widths — in the Smart Relinker preview, use the Col width spinner to widen path columns, or drag the column header dividers directly.
  • Preview before applying — always use Preview Changes before clicking Apply Now. The preview table is fully editable; individual rows can be overridden using the Manual override box.
  • Verify paths — keep the Verify new paths exist checkbox on. If a target file is not found on disk, the tool warns you before writing any changes.
  • Refresh All Lists — click this whenever you open or close workbooks in Excel so the tool's dropdowns stay in sync.
  • profiles.json — edit this file in Notepad to customise quarterly rollover rules. Changes take effect next time you click Refresh All Lists or open the relinker.
